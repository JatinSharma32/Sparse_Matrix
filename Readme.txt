            -:  Sparse Matrix  :-

Basic working: The main work of this program is to take a matrix and 
               check if it is a Sparse matrix or not if yes then the 
               original Matrix will be converted to Sparse Matrix and 
               then it will be printed else it will print the original
               matrix.

Inputs:        -> The Rows And Columns your original Matrix have.
               -> The elements of the matrix (Should be space sepereted).

Output:        -> You will get The Original Matrix and Sparse matrix if 
                  it exists.
               -> Else the Original Matrix will be printed only.
